# Client Research
## Goal
- 클라도 조금 만들어봐야 눈으로도 서버가 제대로 돌아가는지 확인하고, 연습도 될 듯해서 만들어본 프로젝트
- 그런데, 굳이 Unity, Unreal로 눈으로 볼 필요는 없다고 생각했다. 너무 컴퓨터 자원을 많이 먹기도 하고...
- 나중에 1인 개발 할 때 정도야 Unity가 필요하겠지만 굳이 지금 Graphics 를 공부할 필요가 없다고 생각이 든다.
- TCP 환경에서 테스트 할 수 있는 클라 정도는 필요할 듯 하다. (시간이 되면 UDP도)
